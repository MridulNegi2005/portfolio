# Mridul Negi Portfolio

A static, single-page portfolio website for Mridul Negi, a systems-minded software engineer. Built with a focus on clean design, fluid animations, and robust cross-device compatibility.

## Features

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices with fluid typography and dynamic spacing.
- **Custom WebGL Effects:** Includes a touch-tracking WebGL grid lens and subtle background animations for an interactive experience.
- **Scroll-Driven Animations:** Uses modern CSS and vanilla JavaScript for smooth, scroll-triggered visual feedback and element reveals.
- **Cross-Agent Workflow:** Integrated `.ai-sync` directory and context files (CLAUDE.md, GEMINI.md, AGENTS.md) to manage tasks across different AI coding assistants.
- **SEO Ready:** Includes `sitemap.xml` and `robots.txt` for standard search engine indexing.

## Tech Stack

- HTML5
- CSS3 (Custom properties, grid, flexbox, CSS animations)
- Vanilla JavaScript
- WebGL (HTML5 Canvas)

## How to Run

Since this is a static website with no build step required, you can serve it locally using any standard HTTP server.

1. Clone the repository:
   ```bash
   git clone https://github.com/MridulNegi2005/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Start a local server (e.g., using Python):
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`.
